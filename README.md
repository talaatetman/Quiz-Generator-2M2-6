### مولد واجهات الاختبارات الذكية

طريقة الإدخال السهلة:
1. لكى تظهر اسئلة كل دكتور فى قسم مخصص فى واجهة الاختبار المتولد اكتب معرف القسم (مثلاً: dr1) فى مربع الادخال المخصص بها وكذلك اسم القسم (مثلاً: د. أحمد أو اسم الماده وبجانبها اسم الدكتور) وإذا تركت هذه الخانات فارغة سيتم اظهار كل الاسئلة فى قسم واحد عام .
2. اكتب الأسئلة بصيغة: السؤال |*صح | خطأ أو السؤال | صح | *خطأ أو السؤال | خيار 1 | *خيار صح | خيار 3
3. كرر ذلك لكل مادة أو دكتور ، ثم اضغط توليد. يمكنك جمع كل بنك الاسئلة للفصل الدراسى فى اختبار واحد وسيظهر اسئلة كل دكتور على حده.
4. كل مربع ادخال للاسئلة يقبل عدد كبير من الاسئلة بحيث يكون كل سؤال فى سطر وعدم وجود سطور فارغة وعدم وجود فقرات بدون اجابات بعدها.
5. اضغط على زر تنزيل ملف الاختبار أو انسخ الكود المتولد سواء بزر النسخ اعلى الكود او انسخه يدويا وإلصقه فى محرر النصوص واحفظ الملف بإمتداد html مثل (test.html)واحفظه على الجهاز وابدأ فى تشغيل الاختبار.
